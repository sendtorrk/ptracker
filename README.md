# Print Tracker

A hybrid mobile app to track number of pages printed for each billing period.

Powered by:

* Apache Cordova
* Quasar Framework
* Vue.js
* Moment.js

## Install dependencies
```
npm install
```

## Development
```
quasar dev -m ios
```

## Development (launch Xcode)
```
quasar dev -m ios --ide
```

## Release
```
quasar build -m ios
```

## Release (launch Xcode)
```
quasar build -m ios --ide
```

## Debugging 

Open Safari > Develop
