# Print Tracker

## Install the dependencies
```
npm install
```

## Development
```
quasar dev -m ios
```

## Development (launch Xcode)
```
quasar dev -m ios --ide
```

## Release
```
quasar build -m ios
```

## Release (launch Xcode)
```
quasar build -m ios --ide
```

## Debugging 

Open Safari > Develop
